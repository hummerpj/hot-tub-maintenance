## Water Chemistry Targets (with ClearBlue + Ozone)
| Parameter | Target | Notes |
|---|---|---|
| Free Chlorine | 3–5 ppm (can run 1–3 with ClearBlue) | Shock to 12 ppm after heavy use |
| pH | 7.2–7.8 | Borates keep it stable |
| Total Alkalinity | 50 ppm | Key to pH stability |
| Calcium Hardness | 120–150 ppm | Pre-dissolve before adding |
| CYA | 30 ppm | Set at startup with dichlor; switch to bleach at 30 |
| Borates | 50 ppm | Boric acid only; top up if below 30 ppm |
| MgCl₂ | 200–300 ppm | Food-grade flakes only |
| TDS | < 1,500 ppm above baseline | Drain when exceeded |
